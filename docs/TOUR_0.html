A fantastic cereal

Programming languages are user interfaces, more powerfull, composable, modular, scriptable, potentially more complex and users might be specific, but still user interfaces.

Programming is translation from computational substrate language to user / domain language or reverse.

Rye language lives close to the user / domain end of the spectrum.

Some basic elements

  • Rye values and assignment
  • If, either, switch and more
  • Loop and for
  • Map, filter and HOF-s
  • Functions, pure functions and curry

More specific elements

  • Pipe-words, op-words
  • Injected block
  • Validation dialect
  • Kinds and converters
  • Generic methods

Failure handling

  • Failures and errors
  • Check, fix and tidy
  • Returning words
  • Skipping words

Language structures

  • Working with blocks
  • Using contexts
  • Lists and dictionaries
  • JSON and JWT
  • Goroutines and channels

Files, Databases and Web

  • Working with files
  • SQLite and Postgres
  • Serving HTTP and websockets
  • Simple app/API server

More dialects

  • QFrame dialect
  • SAX (XML) like dialect
